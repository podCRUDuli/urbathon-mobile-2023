# Городские связи
На данный момент в большинстве городов не существует единого портала для коммуникации с муниципальными службами, размещения различных новостей, связанных с ЖКХ и т.д.
В рамках задачи участникам предстоит спроектировать и реализовать удобное и эффективное решение для обеспечения прозрачного взаимодействия жителей с муниципальными службами и улучшения жизни населения города.

## Особенности
- Авторизация
- Регистрация
- Страница профиля
- Страница списка новостей
- Страница с деталями новости и опросами
- Страница со списком обращений и комментариями
- Страница с картой, на которой изображены маркеры обращений, 
а также места сбора мусора
- Просмотр своих обращений
- Кроссплатформенность (iOS, Android)
- Страница создания обращения

## Требования
- Node.js (рекомендуется версия 14.x или выше)

## Установка
```bash
git clone https://github.com/podCRUDuli/urbathon-mobile-2023.git
cd urbathon-mobile-2023
npm i
```
Для запуска приложения можете использовать:
```bash
npm run start
```
Это откроет DevTools в вашем браузере. Вы можете запустить приложение на эмуляторе или устройстве, отсканировав QR-код с помощью приложения Expo на своем мобильном устройстве.

Или (требуется компьютер с MacOS, а также установленным на него Xcode)
```bash
npm run ios
```
Или (требуется Android Studio)
```bash
npm run android
```
